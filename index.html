Estrutura Básica do HTML

(Voltar)

A estrutura básica do HTML consiste nas tags 'html', 'head' e 'body', nessa ordem. Para abrir e fechar uma tag, necessitamos colocar os sinais (< >) para abrir e para fechar (<\ >), notar que a diferença é que acrescentamos uma barra no sentido para direita entre os sinais de maior e menor para indicar o fechamento da tag.

Falando sobre Tags

(Voltar)

A tag html consiste na página onde será desenvolvido o código, ou seja, todas as tags estarão dentro dela. Em suma, ela será a página em branco da web sem dados.

A tag head é onde se encontra o nome da página da web, o logo do website, links do CSS, tipos de letras e configurações gerais do site. Há algumas outras tags utilizadas com ela, mas a que mais é utilizada é a tag "title", título da página.

A tag body é onde está toda a estrutura do site em si, é nele que é inserido o texto, imagens, links e tudo que é necessário para o usuário utilizar o website, ou seja, é nele que estão inseridas as mais variadas tags.

Em suma, existem as mais variadas Tags, que podem ser utilizadas de diversas formas e quase todas têm de ter uma abertura e um fechamento, menos alguma com a 'input' e a 'img'.

Atributos das Tags

(Voltar)

Atributos são propriedades que uma tag possa ter. Existem atributos globais, todas as tags podem ter e outras mais específicas que somente certas tags têm.

Entre os atributos globais temos a 'id', cujo objetivo é dar uma identificação a uma tag. Essa identificação é muito utilizada pelo Java para realizar edição nos textos e imagens das tags. Temos também o atributo 'style' que servirá para editar diretamente o texto ou imagem no HTML, ou seja, utiliza comandos da CSS direto no HTML. Por fim, temos o atributo 'class', utilizado com a ligação para edição no CSS, ou seja, as tags com o mesmo atributo class são editadas mais rapidamente pelo CSS.

Os atributos especificos, que são mais utilizados são 'type' com a tag 'input' e o 'src' com a tag 'img'.

Textos

(Voltar)

São as tags aonde vão os textos que os usuários vão ler ao entrar no site. Há diversas tags que podem ser utilizadas nos textos nos arquivos HTML. Temos o 'h', vai do 1 ao 6, 'p', 'blockquote', 'strong', 'u', 'i', 'mark','sup', entre outras.

Segue abaixo o significado de algumas tags mais utilizadas nos textos.

  • Tag 'h' - utilizadas para títulos e subtítulos, sendo que a 'h1' tem a maior letra e vai diminuindo até chegar a 'h6'.
  • Tag 'p' - referente ao parágrafo do texto.
  • Tag 'blockquote' - geralmente usado para citações no texto.
  • Tag 'strong' - utilizado para deixar partes do texto em negrito.
  • Tag 'u' - utilizado para sublinhar partes do texto.
  • Tag 'i' - utilizado para deixar partes do texto em itálico.
  • Tag 'mark' - utilizado para grifar uma parte do texto.
  • Tag 'sup' - utilizado para colocar um subscrito no texto.

As tags acima são somente algumas que são comumente utilizadas há diversas, no site 'w3schools' há uma lista com todas as tags utilizadas nos textos, o link se encontra na referência.

Listas Ordenados e Não Ordenadas

(Voltar)

As Listas Ordenados e Não Ordenadas são utilizadas para organizar alguns dados na página da web ou direcionar para algum site externo ou uma parte específica do próprio site. São utilizadas 2 tags pai, a 'ol', para lista ordenada e a 'ul' para não ordenada, e a tag filha 'li', onde ficam os dados da lista.

Lista ordenada utilizamos a tag 'ol' e a filha 'li', essa tag é exposta em numeral na página web, ou seja, tem uma ordem de prioridade na sua exposição.

Lista não ordenada utilizamos a tag 'ul' e a filha 'li', essa tag teria marcadores não numéricos.

Links

(Voltar)

O link são termos que permiti navegar pela internet. No HTML, os links são tags que permitem navegar entre a página ou fora dela. A tag utilizada é a 'a'.

A tag 'a' é utilizada com o atributo 'href', nele é colocado o link do site, ainda queremos que o usuário visite. E antes da tag 'a' de fechamento, coloca uma breve descrição do que seria o link.

Também utilizamos o atributo 'target', que abrira as opções '_blank', o link abrira em uma nova página, ou a opção '_self', o link abrira na mesma página.

Podemos também utilizar o atributo 'title', que é colocar uma sugestão como 'clique aqui' ou 'acesse aqui', quando se passa o mouse no link.
